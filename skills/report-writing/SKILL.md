# 报告撰写技能

**来源:** K-Dense-AI/scientific-agent-skills (BSD-3-Clause)
**适配:** Hermes Agent Skill 格式
**适用场景:** 研究报告、分析文档、投资备忘录、技术报告

---

## 核心原则

⚠️ **关键原则:**
- **始终用完整段落写作**, 不用项目符号列表作为最终输出
- **两阶段流程:** 先用研究查找创建章节大纲与要点 → 再转换为流畅段落
- **每份报告必须包含数据可视化图表**

---

## 何时使用

- 撰写或修订研究报告的任何部分
- 用IMRAD或其他标准格式构建研究论文
- 格式化引用和参考文献(APA/AMA/Chicago/IEEE)
- 创建、格式化或改进图表和数据可视化
- 撰写符合期刊要求的摘要
- 提高写作清晰度、简洁性和精确性
- 确保正确使用领域特定术语

## 报告结构

### 标准结构 (IMRAD)

**1. 引言 (Introduction)**
- 建立研究问题的重要性
- 系统回顾相关文献
- 识别知识空白或争议
- 陈述清晰的研究问题或假设
- 解释研究的新颖性和意义

**2. 方法 (Methods)**
- 详细研究设计
- 数据来源与处理流程
- 分析方法与工具
- 参数设置与假设

**3. 结果 (Results)**
- 客观呈现发现,不做解释
- 配合图表展示关键数据
- 统计显著性检验结果

**4. 讨论 (Discussion)**
- 解释结果含义
- 与已有研究对比
- 承认局限性
- 提出未来方向

### 投资研究报告结构

**1. 执行摘要**
- 核心结论(1段)
- 关键数据(3-5个要点)
- 建议行动

**2. 市场背景**
- 宏观环境分析
- 行业趋势
- 竞争格局

**3. 标的分析**
- 基本面分析
- 技术面分析
- 估值分析

**4. 风险评估**
- 主要风险因素
- 敏感性分析
- 情景分析

**5. 投资建议**
- 评级与目标价
- 仓位建议
- 止损策略

---

## 写作流程

### 第一阶段: 大纲与要点

```
1. 确定报告类型和目标读者
2. 用研究查找收集关键信息
3. 创建章节大纲
4. 为每个章节列出3-5个要点
5. 确定需要哪些图表/数据
```

### 第二阶段: 段落写作

```
1. 将每个要点扩展为完整段落
2. 确保段落之间有逻辑过渡
3. 插入数据引用和图表
4. 检查术语一致性
5. 润色语言(简洁、精确)
```

## 注意事项

⚠️ **重要提示:**
1. **禁止用项目符号作为最终输出** — 大纲阶段可以用, 成文后必须转为段落
2. **数据必须可追溯** — 每个数据点注明数据来源
3. **图表必须清晰** — 标题、坐标轴标签、图例齐全
4. **结论必须有依据** — 每个判断都要有数据或逻辑支撑
5. **风险提示不可少** — 任何投资建议必须包含风险说明
6. **避免绝对化语言** — 用"可能"、"预计"而非"肯定"、"必然"
