# 五池流转规则

## 流转方向

```
快筛候选池 → 重点观察池 → 持仓池 → （卖出）→ 彻底归档
                ↑
           （满足条件可回归候选池）
```

## 流转条件

### 快筛候选池 → 重点观察池
- 审查评分 ≥ 70
- 驱动逻辑得到新闻/数据验证
- 技术面无明显破位

### 重点观察池
- 评分 ≥ 70 且维持稳定
- 技术形态出现买入信号（放量突破/回踩支撑）
- 大盘环境配合

### 重点观察池 → 决策层（S级操作池）
直接由决策Agent生成执行方案
- 价格触及买入触发条件
- 已确认仓位和止损线

### 持仓池 → 卖出
- 触及止盈目标
- 触及止损线
- 失效条件触发（3日内不启动）

### 任何池 → 边缘池
- 评分 < 60
- 逻辑被证伪
- 出现重大风险事件
- 跌破重要支撑位

## 边缘池说明

边缘池股票来自快筛候选池或重点观察池的审查降级。

**回归条件**（满足以下条件可重新进入候选池审查流程）：
- 消失的风险重新消除
- 新的驱动逻辑出现
- 重新经过审查流程

**彻底淘汰**：边缘池股票超30天无改善 → 彻底归档，移出所有池

## 持仓池卖出后归档

持仓池股票卖出（止盈/止损/失效）后：
- **彻底归档**：不再回到任何池，作为历史记录保存
- 决策日志中标记最终结果（盈利/亏损/持平）
