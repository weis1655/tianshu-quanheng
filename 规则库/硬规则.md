# 天枢 V3 硬规则（绝对不可违反）

## 禁入规则

1. **ST/\*ST股票** — 绝对不买，风险过大
2. **亏损股票** — 连续2年亏损不买
3. **退市风险股** — 有退市风险的股票不买
4. **流通市值<5亿** — 流动性风险过大不买
5. **单日换手率>30%** — 可能有主力出货，不追

## 仓位规则

1. **单票最大仓位**：30%（亏损时绝对不加仓）
2. **单笔最大亏损**：5%（触止损必走）
3. **总仓位上限**：70%（保留子弹）
4. **日内最大亏损**：7%（触及停止操作）
5. **盈利加仓**：只在持仓盈利>5%时才考虑，且最多加原仓位的50%

## 执行规则

1. **T+1限制**：今日买不可今日卖，隔夜必须设止损
2. **止损执行**：触及止损线，无条件执行，不犹豫
3. **失效条件**：3日内不启动，放弃该笔交易
4. **大盘配合**：大盘环境差（跌幅>1%）时，不做新开仓

## 时间规则

1. **早盘前30分钟**：观望为主，不追高
2. **14:50之后**：不新开仓，持仓准备尾盘处理
3. **非交易时间**：只做研究和决策，不执行买卖
