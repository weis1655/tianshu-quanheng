## 2. 功能需求规格

### 2.1 策略池管理（F1）

#### F1.1 多策略注册
- **业务规则**：每个策略有唯一名称，可带描述、标签、版本号
- **参数**：name, enabled, description, allocation, min/max_allocation, max_drawdown, max_position_pct, max_positions, stop_loss, rebalance_threshold, version, tags[], created_at
- **约束**：同名策略不可重复注册；总量不能超过100%分配

#### F1.2 策略启停
- **业务规则**：停用策略保留配置但暂停资金分配；启用的策略进入分配池
- **交互**：`enable_strategy(name)` / `disable_strategy(name)` / `toggle_strategy(name)`

#### F1.3 策略参数配置
- **业务规则**：支持运行时修改参数，修改后自动触发风控重检
- **参数范围校验**：min_allocation ≤ allocation ≤ max_allocation；max_drawdown ≤ -5% 等

#### F1.4 策略版本管理
- **业务规则**：每次参数修改自动递增版本号（major.minor）
- **存储**：保留最近5个版本的配置快照，支持回滚

#### F1.5 单策略运行状态监控
- **监控指标**：当前分配比例、持仓数、收益/亏损、回撤、胜率、夏普、最大连续亏损、运行时（天）
- **状态定义**：`active` / `paused` / `drawdown_warning` / `circuit_triggered` / `stopped`

### 2.2 资金分配（F2）

#### F2.1 固定比例分配
- **算法**：用户指定各策略占比，归一化后分配
- **公式**：`w_i = ratio_i / Σratio_j`，支持 min/max 约束

#### F2.2 风险平价分配
- **算法**：基于策略波动率倒数分配，低波动多配
- **公式**：`w_i = (1/σ_i) / Σ(1/σ_j)`，σ_i 为策略i的波动率
- **波动率估计**：指数加权移动平均（EWMA，λ=0.94），滚动窗口30天

#### F2.3 凯利公式分配
- **算法**：`f* = (p × b - q) / b`，其中 p=胜率, q=1-p, b=平均盈亏比
- **约束**：f* ≤ 40%（半凯利），且不低于 min_allocation，不高于 max_allocation

#### F2.4 按绩效动态调整
- **算法选项**：夏普比率加权 / 胜率加权 / 累计收益加权
- **平滑因子**：`new_allocation = α × signal + (1-α) × old_allocation`，α=0.3
- **调整周期**：每日/每周/每月，可配置

#### F2.5 多层级分配（策略→标的）
- **规则**：策略层分配总资金 → 策略内按评分/信号分配单票仓位
- **公式**：`单票仓位 = 策略分配资金 × (标的评分 / Σ评分) × 最大单票比例`

### 2.3 组合风控（F3）

#### F3.1 策略间相关性监控
- **算法**：计算策略间收益率的皮尔逊相关系数（滚动30天窗口）
- **阈值**：r > 0.7 触发告警，r > 0.85 触发强制再平衡
- **处理**：高相关策略对自动降低总权重

#### F3.2 单策略最大回撤熔断
- **规则**：策略回撤 ≤ max_drawdown（如-15%）→ 自动暂停该策略资金分配
- **恢复**：回撤恢复至阈值一半以上 → 自动恢复
- **熔断状态**：持久化到文件，进程重启后恢复

#### F3.3 组合总仓位约束
- **规则**：总持仓 ≤ MAX_TOTAL_POSITIONS（默认30）；总仓位比例 ≤ MAX_TOTAL_POSITION_PCT（默认100%）
- **排序**：超限时按评分/信心度排序，移除尾部标的

#### F3.4 策略容量上限控制
- **规则**：单策略持仓数 ≤ max_positions；单票仓位 ≤ max_position_pct
- **资金约束**：策略分配资金 ≤ total_capital × max_allocation

#### F3.5 跨策略风险聚合
- **规则**：同一标的出现在多个策略中时，合并计算风险暴露
- **处理**：单标的跨策略总仓位 ≤ 30%（绝对上限）

### 2.4 动态再平衡（F4）

#### F4.1 定期再平衡
- **周期**：每日/每周/每月，可配置
- **执行**：重新计算目标分配 → 执行调仓（生成调仓指令，不下单）

#### F4.2 偏离度触发再平衡
- **规则**：|current_ratio - target_ratio| > rebalance_threshold → 触发
- **阈值**：默认5%，可配置
- **防抖**：触发后1小时内不再重复触发

#### F4.3 平滑调仓
- **规则**：一次调仓不超过总资金M%的变动（默认20%），避免冲击
- **分批**：大额调仓分3天执行

#### F4.4 策略优胜劣汰自动调整
- **规则**：每N天评估一次，淘汰排名末位的策略，释放资金
- **评估指标**：夏普比率（权重50%）+ 胜率（30%）+ 最大回撤（20%，反向）
- **淘汰比例**：倒数1名淘汰，或评分低于阈值

### 2.5 组合绩效（F5）

#### F5.1 组合层面收益
- **指标**：累计收益率、年化收益率、日收益率、波动率
- **计算**：`R_portfolio = Σ(w_i × R_i)`，w_i=策略权重

#### F5.2 风险指标
- **指标**：夏普比率、索提诺比率、卡玛比率、VaR(95%, 1d)、最大回撤
- **计算基准**：无风险利率=2.5%（参考国债收益率）

#### F5.3 归因分析
- **维度**：策略归因、行业归因、风格归因
- **公式**：`R_portfolio = Σ(w_i × R_i) = Σ贡献度_i`

#### F5.4 单策略贡献度拆解
- **指标**：绝对贡献 `= w_i × R_i`；相对贡献 `= w_i × R_i / R_portfolio`
- **可视化**：策略收益贡献饼图、时间序列堆叠图
