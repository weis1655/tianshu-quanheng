## 四、快速上手

```bash
# 批量回测（5个随机策略）
python main.py batch batch --count 5

# 参数扫描
python main.py batch paramscan --param min_score stop_loss_pct

# 查看任务列表
python main.py batch list

# 查看对比排名
python main.py batch compare --id <task_id>

# 生成报告
python main.py batch report --id <task_id>

# 断点续跑演示
python main.py batch resume --fail-at 3
```
