## 五、快速上手

```bash
# 创建TWAP订单
python main.py algo create --algo TWAP --code 600519 --name 茅台 --side buy --qty 10000 --duration 30

# 执行算法
python main.py algo execute --id AE000001

# 查看报告
python main.py algo report --id AE000001

# 运行回放对比
python main.py algo backtest

# 多场景压力测试
python main.py algo scenario
```
