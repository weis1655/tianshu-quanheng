> 日期：2026-07-16 | 状态：✅ 设计完成

## 1. 架构总览

```
条件单系统 → AlgoExecutionEngine → 拆单算法 → 子单 → 行情接口 → 成交回报
                                    ↓
                             ExecutionMonitor → 进度/滑点/偏离度
                                    ↓
                              ExceptionHandler → 异常降级/市价转投
```
