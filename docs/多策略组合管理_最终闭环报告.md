## 五、上线操作指南

### 5.1 快速开始

```bash
# 1. 注册策略
python main.py portfolio register --name "趋势跟踪" --allocation 0.3
python main.py portfolio register --name "均值回归" --allocation 0.2

# 2. 查看策略列表
python main.py portfolio list

# 3. 执行资金分配
python main.py portfolio allocate --method risk_parity

# 4. 执行再平衡
python main.py portfolio rebalance --method risk_parity

# 5. 组合风控检查
python main.py portfolio risk

# 6. 生成绩效报告
python main.py portfolio report

# 7. 查看组合状态（默认）
python main.py portfolio
# 或
python main.py 组合管理
```

### 5.2 管理命令

| 命令 | 说明 |
|:-----|:------|
| `portfolio register --name X --allocation 0.3` | 注册策略 |
| `portfolio enable --name X` | 启用策略 |
| `portfolio disable --name X` | 停用策略 |
| `portfolio toggle --name X` | 切换启停 |
| `portfolio update --name X --allocation 0.4` | 更新参数 |
| `portfolio status --name X` | 查看策略详细状态 |
| `portfolio history --name X` | 版本历史 |
| `portfolio rollback --name X --version 1.0` | 回滚版本 |
| `portfolio remove --name X` | 移除策略 |
| `portfolio allocate --method risk_parity` | 资金分配预览 |
| `portfolio rebalance --method kelly` | 执行再平衡 |
| `portfolio periodic` | 定期再平衡检查 |
| `portfolio survival` | 优胜劣汰 |
| `portfolio risk` | 组合风控检查 |
| `portfolio report` | 生成绩效报告 |

### 5.3 配置示例（config.yaml）

```yaml
portfolio:
  enabled: true
  default_allocation_method: "risk_parity"
  rebalance:
    periodic: true
    periodic_interval: "weekly"
    periodic_day: "Monday"
    deviation_threshold: 0.05
    max_turnover_per_trade: 0.20
  risk:
    max_total_positions: 30
    correlation_threshold_warning: 0.7
    correlation_threshold_force: 0.85
    cross_strategy_position_limit: 30.0
  performance:
    risk_free_rate: 2.5
    survival_competition_days: 30
    survival_elimination_count: 1
```

### 5.4 完整使用示例

```bash
# 初始化2个策略
python main.py portfolio register --name "趋势跟踪" --allocation 0.3 --desc "基于均线突破"
python main.py portfolio register --name "均值回归" --allocation 0.3 --desc "超跌反弹"
python main.py portfolio list

# 分配资金
python main.py portfolio allocate --method risk_parity

# 查看详情
python main.py portfolio status --name "趋势跟踪"

# 更新参数
python main.py portfolio update --name "趋势跟踪" --max-dd -12.0 --max-pos 3

# 检查熔断
python main.py portfolio risk

# 生成绩效报告
python main.py portfolio report
```
