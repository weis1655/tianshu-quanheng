## 五、架构说明

```
main.py event → event_engine.py → 扫描全A股 → 事件检测器 → 事件评分 → 信号报告
                                       ↓
                                event_backtest.py → 回测引擎 → 多策略对比
                                       ↓
                                EventPoolBridge → 候选池格式 → 五池接入
```

## 六、上线操作指南

```bash
# 事件扫描（默认）
python main.py event

# 查看事件列表
python main.py event list

# 评分指定股票
python main.py event score --code 600519

# 回测模拟（300条事件）
python agents/event_backtest.py backtest --events 300

# 信号报告生成
python agents/event_backtest.py signal
```
