## 四、快速上手

```bash
# 创建演示条件单集
python main.py order demo

# 查看条件单列表
python main.py order list

# 扫描一次条件单（触发符合条件的）
python main.py order scan

# 创建止损单
python main.py order create --type FIXED_SL --code 600519 --name 茅台 --trigger 1425 --cost 1500

# 创建分级止盈
python main.py order create --type TIERED_TP --code 600519 --name 茅台 --cost 1500

# 查看触发日志
python main.py order log

# 撤销条件单
python main.py order cancel --id CO000001
```
