## 2. 条件单类型

### 2.1 价格条件单（P0）

| 类型 | 触发条件 | 执行动作 | 业务场景 |
|:-----|:---------|:---------|:---------|
| 限价买入(LIMIT_BUY) | 最新价 ≤ 触发价 | 买入N股/元 | 回调到支撑位买入 |
| 限价卖出(LIMIT_SELL) | 最新价 ≥ 触发价 | 卖出N股/元 | 反弹到目标价卖出 |
| 突破买入(BREAK_BUY) | 最新价 ≥ 触发价 | 买入N股/元 | 突破压力位追入 |
| 跌破卖出(BREAK_SELL) | 最新价 ≤ 触发价 | 卖出N股/元 | 跌破支撑位止损 |
| 网格买入(GRID_BUY) | 价格下跌X% | 分批买入 | 网格交易 |

### 2.2 止盈止损条件（P0）

| 类型 | 触发条件 | 执行动作 | 说明 |
|:-----|:---------|:---------|:------|
| 固定止损(FIXED_SL) | 最新价 ≤ 成本×(1-sl%) | 全部卖出 | 单票最大亏损控制 |
| 分级止盈(TIERED_TP) | 价格达到N个目标档位 | 分批卖出 | 每档卖1/3或1/2 |
| 移动止损(TRAILING_SL) | 价格从高点回撤X% | 全部卖出 | 锁定利润+控制回撤 |
| 时间止损(TIME_SL) | 持仓超N天未达目标 | 全部卖出 | 资金效率管理 |

### 2.3 盈亏条件（P0）

| 类型 | 触发条件 | 执行动作 | 说明 |
|:-----|:---------|:---------|:------|
| 单日亏损降仓(DAILY_LOSS) | 单日浮亏>X% | 减仓50% | 风险控制 |
| 单票回撤平仓(DRAWBACK) | 累计亏损>X% | 全部卖出 | 最大回撤控制 |
| 组合回撤全减(PORTFOLIO_DD) | 总回撤>X% | 全部减仓 | 系统性风险 |

### 2.4 高级策略单（P1）

| 类型 | 触发条件 | 执行动作 | 说明 |
|:-----|:---------|:---------|:------|
| 分批建仓(DCA) | 价格每跌X%加仓 | 买入固定金额 | 均价策略 |
| 条件暂停(PAUSE) | 市场异常触发 | 暂停所有新开仓 | 风控暂停 |

## 3. 优先级规则

| 优先级 | 条件类型 | 说明 |
|:------:|:---------|:------|
| 1(最高) | 风控条件 | 单日亏损降仓/组合回撤全减 — 资金安全第一 |
| 2 | 止损条件 | 固定止损/移动止损/时间止损 — 保住本金 |
| 3 | 止盈条件 | 分级止盈 — 兑现利润 |
| 4 | 价格条件 | 限价/突破/跌破 — 正常交易 |
| 5(最低) | 高级策略 | 网格/DCA — 策略性建仓 |

## 4. 状态机

```
CREATED → ACTIVE → TRIGGERED → EXECUTING → EXECUTED
  ↓          ↓         ↓                       ↓
CANCELLED  PAUSED    FAILED               PARTIAL
  (可撤销)   (暂停)   (触发失败)             (部分成交)
```
