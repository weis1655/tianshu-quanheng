## 3. 数据模型

### 3.1 事件数据结构

```python
@dataclass
class EventRecord:
    event_id: str                    # 事件唯一标识
    event_type: str                  # 事件类型代码 (EV-01~EV-14)
    event_name: str                  # 事件名称
    code: str                        # 股票代码
    name: str                        # 股票名称
    trigger_date: str                # 事件触发日期
    event_score: float               # 事件强度评分 (0-100)
    event_data: Dict[str, Any]       # 事件原始数据
    factors: Dict[str, float]        # 事件因子
    status: str                      # active / expired / executed
```

### 3.2 事件因子

```python
@dataclass 
class EventFactors:
    intensity_score: float           # 事件强度 (0-100)
    expected_surprise: float         # 预期差 (实际vs预期)
    post_event_return: float         # 事件后N日收益规律
    decay_period: int                # 衰减周期（天）
    win_rate_hist: float             # 历史胜率
    avg_return_hist: float           # 历史平均收益
    sharpe_hist: float               # 历史夏普
```

### 3.3 策略配置

```python
@dataclass
class EventStrategyConfig:
    event_type: str                  # 事件类型
    enabled: bool                    # 是否启用
    min_score: float                 # 最低事件评分
    max_position_pct: float          # 最大仓位
    stop_loss_pct: float             # 止损
    take_profit_pct: float           # 止盈
    hold_days: int                   # 持仓天数
    entry_timing: str                # 入场时机: open/close/next_open
    require_volume_confirm: bool     # 是否需要成交量确认
```

## 4. 事件评分体系

### 4.1 评分维度

| 维度 | 权重 | 说明 |
|:-----|:----:|:------|
| 事件强度 | 30% | 事件本身的置信度与幅度 |
| 基本面 | 25% | 营收/利润/ROE综合质量 |
| 技术面 | 25% | 价格位置/趋势/成交量 |
| 市场环境 | 20% | 大盘趋势/板块热度 |

### 4.2 评分公式

```
EventScore = Intensity × 0.30 + Fundamental × 0.25 + Technical × 0.25 + Market × 0.20
```

### 4.3 衰减模型

```
Score_衰减(t) = Score_初始 × max(0, 1 - t / decay_period)
```
其中 t = 事件触发后的天数，decay_period 默认为5/10/20天
